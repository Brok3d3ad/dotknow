SVG Processor Tool
Version: 1.0.0
Release Date: 2024-03-15

This release includes:
- Stable Windows executable for SVG to JSON conversion
- Support for all major SVG element types (rectangles, circles, ellipses, lines, polylines, polygons, paths, text elements)
- Ignition SCADA project export functionality
- Configurable settings for element type and properties path
- Sample SVG file for testing purposes
